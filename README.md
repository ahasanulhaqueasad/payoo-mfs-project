# Payoo MFS - Digital Wallet Login

**Payoo MFS** is a simple mobile financial service (MFS) web interface built using **HTML, Tailwind CSS, DaisyUI, and JavaScript**.
This project demonstrates a secure login interface for a digital wallet system.

## 🚀 Features

* Modern Login UI with Glassmorphism Design
* Account Number Input
* 4 Digit Secure PIN Input
* Responsive Layout
* Built with Tailwind CSS & DaisyUI
* JavaScript Login Validation
* Clean and Professional UI

## 🛠 Technologies Used

* HTML5
* Tailwind CSS
* DaisyUI
* JavaScript (DOM Manipulation)

## 🔐 Login System

The login system checks the **Account Number** and **PIN** using JavaScript.
If the correct credentials are entered, the user will be redirected to the home page.

Example Logic:

```
Account Number: 123456789
PIN: 1234
```

## ▶ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
